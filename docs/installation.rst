Installation
============

Requirements
-----------

- Python 3.8+
- pydantic 2.0+
- openai 1.99.6+

Installing from PyPI
-------------------

.. code-block:: bash

    pip install etracer

Development Installation
----------------------

To install etracer with development dependencies:

.. code-block:: bash

    # Clone the repository
    git clone https://github.com/emmanuelkasulani/etracer.git
    cd etracer

    # Install development dependencies
    pip install -e ".[dev]"
